◆ generatePredicate()

std::function< bool(Value &)> AANonNullImpl::generatePredicate ( Attributor &  A)

Generate a predicate that checks if a given value is assumed nonnull.

The generated function returns true if a value satisfies any of following conditions. (i) A value is known nonZero(=nonnull). (ii) A value is associated with AANonNull and its isAssumedNonNull() is true.
