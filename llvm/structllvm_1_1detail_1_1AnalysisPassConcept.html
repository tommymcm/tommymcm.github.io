Detailed Description

template<typename IRUnitT, typename PreservedAnalysesT, typename InvalidatorT, typename... ExtraArgTs>
struct llvm::detail::AnalysisPassConcept< IRUnitT, PreservedAnalysesT, InvalidatorT, ExtraArgTs >

Abstract concept of an analysis pass.

This concept is parameterized over the IR unit that it can run over and produce an analysis result.
