Detailed Description

Base class for all errors indicating malformed binary files.

Having a subclass for all malformed binary files allows archive-walking code to skip malformed files without having to understand every possible way that a binary file might be malformed.

Currently inherits from ECError for easy interoperability with std::error_code, but this will be removed in the future.
